# Kerodex demo vehicle photos

The `models/` directory contains locally bundled Wikimedia Commons vehicle
photos selected for each demo listing's year, make, and model. Full source,
creator, and license metadata is stored in `ATTRIBUTION.json`.

The body-type folders contain app-owned generated photos used only by the Buyer
Guide shape cards. Demo disclosure remains in the Kerodex UI rather than inside
any vehicle photo.
